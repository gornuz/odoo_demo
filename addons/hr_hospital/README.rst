HR Hospital
===========

Hospital management basics: doctors, patients, diseases, and visits.

Installation
------------

- Add the module to your addons path.
- Update apps list and install the module.

Configuration
-------------

No additional configuration is required.

Usage
-----

- Open the Hospital menu.
- Manage doctors, patients, diseases, and visits.

-----------
